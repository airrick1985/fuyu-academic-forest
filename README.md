# 富宇學森 The Academic Forest

> 建案電子銷售表板 — 學境寓所．校園共森

## 簡介

專為銷售人員設計的建案數位展示工具，以 50 吋以上大電視（4K、16:9）為主要展示載體，透過高質感視覺設計與流暢動畫，輔助銷售人員向客戶介紹「富宇學森」建案。

## 技術

- 純 HTML5 + CSS3 + Vanilla JavaScript
- GitHub Pages 部署
- Canvas API 螢火蟲粒子特效
