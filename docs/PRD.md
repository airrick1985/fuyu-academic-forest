## 1. 產品概述

### 1.1 產品名稱
**富宇學森 The Academic Forest — 建案電子銷售表板**

### 1.2 產品定位
專為銷售人員設計的建案數位展示工具，以 50 吋以上大電視（4K、16:9）為主要展示載體，透過高質感視覺設計與流暢動畫，輔助銷售人員向客戶介紹「富宇學森」建案。

### 1.3 Slogan
> **學境寓所．校園共森**

### 1.4 核心原則
- 🎯 **純展示型**：此為銷售工具，非客戶自助瀏覽頁面
- 🚫 **無互動表單**：不需要聯絡方式、聯絡我們、註冊等功能
- 🖥️ **大螢幕優先**：針對 50 吋+ 4K 電視優化（3840×2160, 16:9）
- 🌿 **自然 × 學術 × 高端 × 靜謐**：整體設計風格關鍵字

## 2. 目標用戶與使用場景

### 2.1 目標用戶
| 角色 | 說明 |
|------|------|
| **銷售人員** | 在接待中心操作電腦，透過大螢幕展示建案資訊給客戶 |

### 2.2 使用場景
1. 客戶到訪接待中心，銷售人員開啟電子表板
2. 首頁 Hero 自動播放森林意象與螢火蟲特效，營造氛圍
3. 銷售人員依需求點擊 Top Bar 導航至不同頁面進行介紹
4. 品牌介紹頁面可展示富宇機構實力與建築團隊陣容

### 2.3 使用環境
| 項目 | 規格 |
|------|------|
| **螢幕尺寸** | 50 吋以上大電視 |
| **解析度** | 4K（3840 × 2160） |
| **比例** | 16:9 |
| **瀏覽器** | Chrome / Edge（最新版） |
| **操作方式** | 銷售人員透過電腦滑鼠/鍵盤操作 |

## 4. 功能需求

### 4.1 全域 Top Bar

#### 4.1.1 基本規格
| 項目 | 說明 |
|------|------|
| **定位** | `position: fixed` 永遠固定於頂部 |
| **層級** | `z-index: 9999` 確保在最上層 |
| **高度** | 約 80px（4K 下視覺適配） |
| **背景** | 深色半透明毛玻璃效果（glassmorphism） |
| **適用範圍** | 所有頁面共用（統一元件） |

#### 4.1.2 導航項目

| 序號 | 項目 | 行為 | 子選單 |
|------|------|------|--------|
| 1 | 建案 LOGO（圖檔） | 點擊返回首頁 `index.html` | 無 |
| 2 | 品牌介紹 | Hover 展開子選單 | ✅ 見下方 |
| 3 | 地段環境 | 點擊進入「即將推出」頁 | 無 |
| 4 | 建築規劃 | 點擊進入「即將推出」頁 | 無 |
| 5 | 建材設備 | 點擊進入「即將推出」頁 | 無 |
| 6 | 工法介紹 | 點擊進入「即將推出」頁 | 無 |

#### 4.1.3 品牌介紹子選單
| 子項目 | 連結 |
|--------|------|
| 富宇機構 | `brand-fuyu.html` |
| 建築團隊 | `brand-team.html` |

> **建築團隊頁面內部**再以區塊劃分：建築設計、景觀公設、燈光設計

#### 4.1.4 互動效果
- Hover 時導航文字發光/底線動畫
- 品牌介紹的子選單以淡入下滑方式展開
- 當前所在頁面的導航項目高亮顯示
- 滑動時 Top Bar 背景加深毛玻璃效果

### 4.2 首頁 Hero（index.html）

#### 4.2.1 視覺組成
| 元素 | 說明 |
|------|------|
| **背景** | 全屏明亮森林意象圖片（AI 生成） |
| **中央 LOGO** | 建案 LOGO 圖檔（使用者提供） |
| **Slogan** | `學境寓所．校園共森` 顯示於 LOGO 下方 |
| **螢火蟲特效** | Canvas 粒子動畫，金色/綠色光點漂浮 |
| **遮罩** | 半透明漸層遮罩，確保文字可讀性 |

#### 4.2.2 動畫效果
| 動畫 | 說明 |
|------|------|
| **LOGO 入場** | 頁面載入後，LOGO 由透明淡入 + 微幅放大，搭配柔和光暈 |
| **Slogan 入場** | LOGO 動畫結束後，Slogan 淡入顯現 |
| **螢火蟲粒子** | 持續播放，光點隨機漂浮、明暗閃爍、大小不一 |
| **背景微動** | 森林背景緩慢微幅放大（Ken Burns 效果），增加生命感 |

#### 4.2.3 螢火蟲粒子規格
| 參數 | 值 |
|------|-----|
| 粒子數量 | 60-100 顆 |
| 顏色 | 金色 `#FFD700`、暖黃 `#F5E6A8`、淡綠 `#A8E6A0` |
| 大小 | 2px - 8px 隨機 |
| 移動速度 | 極緩慢（0.2-0.8px/frame） |
| 明暗效果 | opacity 在 0.2-1.0 之間週期性變化 |
| 光暈效果 | 使用 `box-shadow` 或 `radial-gradient` 模擬發光 |

#### 4.3.2 設計要點
- 背景延續森林意象（淡化/模糊處理）
- 大字標題搭配優雅排版
- 段落文字使用較大字級（4K 螢幕適配）
- 區塊之間使用滾動淡入動畫
