# 🔧 如果 F12 還是無法打開

## 快速檢查清單

### ✅ 方案 1: 強制打開 DevTools

在命令行執行 EXE 時加參數：

```bash
cd dist
.\富宇學森-1.0.0-portable.exe --inspect=9222
```

這會強制開啟調試模式，之後可以用 F12。

### ✅ 方案 2: 使用其他快捷鍵

如果 F12 不行，試試：
- **Ctrl + Shift + I** （打開 DevTools）
- **Ctrl + Shift + J** （打開 Console）
- **Ctrl + Shift + C** （選擇元素）

### ✅ 方案 3: 修改 main.js 自動打開

如果以上都不行，我們可以改回自動打開：

編輯 `electron/main.js` 第 81-84 行：

```javascript
// 改成這樣（自動打開）
mainWindow.webContents.openDevTools();
```

然後重新打包：
```bash
npm run build
npm run electron-build-win
```

### ✅ 方案 4: 檢查應用是否真的啟動了

即使沒有 DevTools，應用應該能正常運行。

試試：
1. 雙擊 EXE
2. 點擊各個菜單和頁面
3. 檢查功能是否正常

如果功能正常，但只是無法看到 DevTools，那還是可以的。
